# csharp-video-size-calculator
C# Video Size Calculator that pre calculates before starting record according to record format, bitrate and audio channels, sample rates etc.

## Calculation Options
	-Video Format
	-Video Frame per Second
	-Audio Channels
	-Audio Sample Rate
	-Audio Bit Depth
